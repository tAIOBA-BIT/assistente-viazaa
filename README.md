# 📖 Sobre o Projeto

O **Assistente Inteligente VIAZA** é um aplicativo desktop desenvolvido para automatizar tarefas operacionais relacionadas ao processo de emissão de passagens aéreas.

O sistema atua como um assistente do operador, monitorando Ordens de Produção (OPs), identificando automaticamente retarificações, pesquisando tarifas em diferentes portais, comparando valores, gerando documentação e automatizando a comunicação entre os setores envolvidos.

O objetivo principal é reduzir o tempo operacional, diminuir erros humanos e padronizar todos os processos realizados durante uma emissão.

---

# 🎯 Objetivos

O projeto busca automatizar atividades repetitivas executadas diariamente pelos operadores, permitindo que eles concentrem seu trabalho apenas nas decisões que realmente exigem intervenção humana.

Entre os principais objetivos estão:

- Monitoramento automático de OPs
- Identificação de retarificações
- Pesquisa automática em múltiplos portais
- Comparação inteligente de tarifas
- Identificação do melhor preço disponível
- Geração automática de documentos
- Comunicação automatizada via WhatsApp
- Envio de e-mails ao cliente
- Registro completo das operações
- Dashboard de produtividade
- Histórico de consultas
